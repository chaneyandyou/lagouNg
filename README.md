## 使用Angular.js开发一个拉钩webapp简易版
[项目演示](http://chaneychan.com/lagouNg/dist/index.html#!/main)

## 配置
* 将项目``clone``到本地

```
# 定位到任意目录
$ cd path/to/root

# 克隆仓库到指定的文件夹
$ git clone https://github.com/chaneyandyou/lagouNg.git

# 进入指定的文件夹
$ cd [project-name]
```

* 基于``Node``环境，使用bower包管理工具

```
$ npm install
$ bower install
```
* 自动化构建工具使用``gulp``

```
//全局安装gulp
$ gulp install -g

//启动服务
$ gulp serve
```
